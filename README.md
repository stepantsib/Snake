# 🐍 Snake Game

Классическая игра "Змейка" на Python с использованием библиотеки Pygame.

## 📝 Описание

Простая версия змейки:
- Постепенное увеличение скорости
- Счётчик очков
- Обработка столкновений
- Чистая архитектура (разделение логики и инфраструктуры)

## 🎮 Управление

- **↑**  изменить направление (вверх)
- **↓**  изменить направление (вниз)
- **←**  изменить направление (влево)
- **→**  изменить направление (вправо)

> Змейка не может резко разворачиваться на 180° (запрещён разворот в противоположную сторону).
