# 🐍 Snake Game

Классическая игра "Змейка" на Python с использованием библиотеки Pygame.

## 📝 Описание

Простая версия змейки:
- Постепенное увеличение скорости
- Счётчик очков
- Обработка столкновений
- Чистая архитектура (разделение логики и инфраструктуры)

## 🎮 Управление

- **↑**  изменить направление (вверх)
- **↓**  изменить направление (вниз)
- **←**  изменить направление (влево)
- **→**  изменить направление (вправо)

> Змейка не может резко разворачиваться на 180° (запрещён разворот в противоположную сторону).

## 🚀 Как запустить игру

Для запуска игры вам потребуется установленный Python (версия 3.13 или выше). 

1. Cкачайте файлы игры в отдельную папку.
2. Откройте терминал и перейдите в папку с игрой.
3. Установите библиотеку Pygame с помощью пакетного менеджера uv:
   ```bash
   uv add pygame

4. Запустите главный файл игры:
    ```bash
    python main.py
   
## 🛠 Консоль разработчика и Чит-коды

В игре предусмотрена скрытая консоль, которая позволяет вводить чит-коды. Чтобы открыть её во время прохождения уровня, нажмите клавишу **`~`** (тильда). 

Игра автоматически поставится на паузу, а в левом верхнем углу появится строка ввода. Введите один из чит-кодов ниже и нажмите **Enter**, чтобы применить эффект.

| Чит-код | Описание эффекта                                                                                                               |
| :--- |:-------------------------------------------------------------------------------------------------------------------------------|
| `godmode` | **Режим бога:** Змейка становится бессмертной. Вы можете спокойно проходить сквозь стены, препятствия и даже собственное тело. |
| `speedhack` | **Speedhack:** Устанавливает максимальную скорость движения змейки (эффект действует до повторного ввода команды).             |
| `grow` | **Мгновенный рост:** Добавляет 5 блоков длины к хвосту змейки. Полезно для быстрого набора очков.                              |
| `tiny` | **Микро-змейка:** Сбрасывает длину хвоста, оставляя только голову (длина = 1). Идеально, если вы застряли в узком коридоре.    |
| `nextlevel` | **Пропуск уровня:** Мгновенно засчитывает победу на текущем этапе и переводит вас на следующий уровень (или к экрану финала).  |

> **Примечание:** Повторный ввод чита `godmode` отключает режим бессмертия.
